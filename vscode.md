# 常用快捷键

## 侧边栏

檔案管理（ctrl + shift + e )

搜索（ctrl + shift + f）

Git（ctrl + shift + g）

插件（ctrl + shift + x）

调试（ctrl + shift + d）

命令行（ctrl + ‵）

## 在編輯器之間切換

這個快速鍵不論在何處都可以快速切換到編輯器的群組，譬如從整合終端機或是側邊欄
ctrl + [1-9]
在群組內快速切換分頁
alt + [1-9]
ctrl + tab
控制分頁在群組之間移動
alt + ←→

## 多重游標操作

alt + shift + drag
